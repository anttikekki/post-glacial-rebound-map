- [GLARE materials](https://drive.google.com/drive/folders/184nPIZuX83gr3Yd6tVBGXCkpUysNY-CO) in Google Drive.
- [land-uplift-recons](https://github.com/Hakonaki/land-uplift-recons) reposity on GitHub.
